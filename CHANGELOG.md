# Changelog

## 0.1.0 — unreleased

### Added

- Core context-sync protocol (HTTP server + SQLite storage).
- JavaScript SDK (`agentmail`).
- Python SDK (`agentmail` on PyPI), async + sync wrapper.
- MCP adapter (`agentmail-mcp`) exposing the protocol as MCP tools.
- CC / BCC / ReplyAll multi-agent threads.
- Optional API-key auth via `AGENTMAIL_API_KEY`.
- Vitest test suite for JS, pytest suite for Python.
- GitHub Actions CI matrix.
- Research+Writer demo app showing cold-restart context recovery.
